# Portafolio

Este proyecto es una página web en el que mostrar mi portafolio con proyectos importantes realizados en el bootcamp de TripleTen, cuenta con una sección de presentación que incluye enlaces al CV, github y contacto.
Los proyectos se organizan en tarjetas, cada uno tiene un enlace para ver la demo en vivo y el repositorio en GitHub.
Finalmente, el área de contacto incluye enlace directo del correo electrónico, así como a las redes sociales.

Todos los enlaces de navegación tienen un aspecto hover.
Para este proyecto se generaron 3 media queries para pantallas de 1440 px (código base), una intermedia de 768px y una chica de 320px, para mejorar el diseño responsivo, la primera sección se organizó con grid.
